Arduino School Project

Arduino Hardware C++ DIY Smart Devices

An Arduino-based school project focused on creating assistive technology for the visually impaired. The project includes two main components: Smart Safety Goggles and a Smart Walking Stick, both designed to help detect obstacles and improve mobility.

Project Components

Smart Safety Goggles

The Smart Safety Goggles use ultrasonic sensors to detect obstacles at eye level and above, helping visually impaired users navigate their environment safely.

Features:

  • Ultrasonic sensor for obstacle detection
  • Vibration motor for haptic feedback
  • Arduino-controlled processing
  • Portable and lightweight design

Project Demonstration:

Arduino Code:

// DIY smart glass for blind
// using by ultrasonic sensor
// Created by Technical idEas (yt)

#define trigPin1 5
#define echoPin1 6
#define motor1 7

long duration1;
int distance1;

void setup() {
  pinMode(trigPin1, OUTPUT);
  pinMode(echoPin1, INPUT);
  pinMode(motor1, OUTPUT);
  
  Serial.begin(9600);
}

void loop() {
  // Obstacle detection for sensor 1
  digitalWrite(trigPin1, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin1, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin1, LOW);
  
  duration1 = pulseIn(echoPin1, HIGH);
  distance1 = duration1 * 0.034 / 2;
  
  Serial.print("Distance1: ");
  Serial.println(distance1);
  
  // Activate motor if obstacle is detected within threshold
  if (distance1 < 100) {
    digitalWrite(motor1, HIGH);
  } else {
    digitalWrite(motor1, LOW);
  }
  
  delay(100);
}

Smart Walking Stick

The Smart Walking Stick is designed to detect obstacles at ground level, enhancing mobility for visually impaired individuals by providing early warnings about potential hazards.

Features:

  • Multiple ultrasonic sensors for wide-range detection
  • Buzzer for audible alerts
  • Vibration feedback system
  • Durable construction for daily use

Project Demonstration:

Arduino Code:

// Smart Walking Stick for the Visually Impaired
// Using multiple ultrasonic sensors

#define trigPin1 9
#define echoPin1 10
#define trigPin2 11
#define echoPin2 12
#define buzzer 8
#define vibMotor 7

long duration1, duration2;
int distance1, distance2;

void setup() {
  pinMode(trigPin1, OUTPUT);
  pinMode(echoPin1, INPUT);
  pinMode(trigPin2, OUTPUT);
  pinMode(echoPin2, INPUT);
  pinMode(buzzer, OUTPUT);
  pinMode(vibMotor, OUTPUT);
  
  Serial.begin(9600);
}

void loop() {
  // Measure distance from first sensor
  distance1 = getDistance(trigPin1, echoPin1);
  
  // Measure distance from second sensor
  distance2 = getDistance(trigPin2, echoPin2);
  
  // Output distances to serial monitor
  Serial.print("Distance1: ");
  Serial.print(distance1);
  Serial.print(" cm, Distance2: ");
  Serial.print(distance2);
  Serial.println(" cm");
  
  // Alert if obstacles detected by either sensor
  if (distance1 < 80 || distance2 < 80) {
    // Closer obstacles trigger faster alerts
    int delayTime = map(min(distance1, distance2), 0, 80, 100, 500);
    alertUser(delayTime);
  } else {
    // No obstacles detected
    noTone(buzzer);
    digitalWrite(vibMotor, LOW);
  }
  
  delay(100);
}

// Function to measure distance using ultrasonic sensor
int getDistance(int trigPin, int echoPin) {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  long duration = pulseIn(echoPin, HIGH);
  int distance = duration * 0.034 / 2;
  
  return distance;
}

// Function to alert user with buzzer and vibration
void alertUser(int delayTime) {
  tone(buzzer, 1000);
  digitalWrite(vibMotor, HIGH);
  delay(delayTime);
  noTone(buzzer);
  digitalWrite(vibMotor, LOW);
  delay(delayTime);
}

Project Details

Components Used

  • Arduino Uno Board
  • Ultrasonic Sensors (HC-SR04)
  • Vibration Motors
  • Buzzer
  • Jumper Wires
  • 9V Batteries

Functionality

The system detects obstacles using ultrasonic sensors and provides feedback through vibration motors and audio signals. The Smart Goggles focus on obstacles at head level, while the Smart Stick detects ground-level obstacles, creating a comprehensive mobility solution.

Learning Outcomes

  • Arduino programming in C/C++
  • Circuit design and implementation
  • Working with sensors and actuators
  • Problem-solving for real-world applications
  • Designing assistive technology
